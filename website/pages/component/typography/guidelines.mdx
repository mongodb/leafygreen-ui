<Subtitle>Links</Subtitle>
<Body>
  The most common usage of a link is inline within body copy. When this is the

  tab/window use a “new tab/window” icon to indicate that the link opens
  somewhere new. If the link redirects on the same page the user is on there is
  no need to append the icon.
