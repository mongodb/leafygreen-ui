# @lg-tools/update

Update tools for LeafyGreen

## Usage

Updates all installed LeafyGreen packages.

```
> lg update
```

### Options

#### `--scope`

Select the scope of packages to update. Some valid scopes are:

- `@leafygreen-ui`
- `@lg-tools`

#### `--latest`

Uses `yarn upgrade --latest`, and ignores package versions
