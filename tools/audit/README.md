# Audit

![npm (scoped)](https://img.shields.io/npm/v/@leafygreen-ui/audit.svg)

#### [View on MongoDB.design](https://www.mongodb.design/component/audit/live-example/)

## Usage

```bash
lg audit
```

or with (p)npx

```bash
pnpx @lg-tools/cli audit
```

### Options

- `[directory]`: The directory of the codebase (defaults to current directory)
- `-o`: name of the output file (e.g. `my-report.md`)
- `--no-write`: Skips writing the output file

Example:

```bash
pnpx @lg-tools/cli audit ~/path/to/my/codebase -o my-report.md
```
