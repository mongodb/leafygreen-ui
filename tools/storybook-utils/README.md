## Installation

### Yarn

```shell
yarn add @lg-tools/storybook-utils
```

### NPM

```shell
npm install @lg-tools/storybook-utils
```

## Usage

`@lg-tools/storybook-utils` provides several utilities & types to help write Storybook files
