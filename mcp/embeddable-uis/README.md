# Embeddable UIs

## Installation

### PNPM

```shell
pnpm add @lg-mcp/embeddable-uis
```

### Yarn

```shell
yarn add @lg-mcp/embeddable-uis
```

### NPM

```shell
npm install @lg-mcp/embeddable-uis
```

## Overview

`@lg-mcp/embeddable-uis` provides embeddable UI components for MCP (Model Context Protocol) integrations.
