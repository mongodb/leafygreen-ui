# SortableChartContainer

Adds drag and drop sorting for `Chart` and `ChartGroup` components.

![npm (scoped)](https://img.shields.io/npm/v/@lg-charts/sortable-chart-container.svg)

## Installation

### Yarn

```shell
yarn add @lg-charts/sortable-chart-container
```

### NPM

```shell
npm install @lg-charts/sortable-chart-container
```
